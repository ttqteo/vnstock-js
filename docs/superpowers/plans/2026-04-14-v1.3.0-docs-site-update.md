## Task 4: Rewrite RealtimeQuote docs for v2 API

**Files:**
- Overwrite: `contents/docs/data/RealtimeQuote/index.mdx`

**Step 1: Read current content** — expect pre-v1.1 API (`vnstock.realtime.connect({onMessage})`).

**Step 2: Replace with v1.3.0 `RealtimeClient` EventEmitter API**

Structure:
- Intro — what realtime is, which data source (SSI iBoard)
- API — `realtime.create({ symbols })` returns `RealtimeClient`
- Events: `connected`, `quote`, `disconnected`, `reconnecting`, `error`
- Methods: `connect()`, `subscribe(symbols)`, `unsubscribe(symbols)`, `disconnect()`
- Options: `symbols`, `autoReconnect`, `reconnectInterval`, `maxReconnectAttempts`, `deadManTimeout`
- Full working example
- Note about browser vs Node WebSocket (hidden from user, lib handles it)
- Trading hours note (9:00-11:30, 13:00-14:45 VN)

Example code block:

```mdx
\`\`\`ts
import { realtime } from 'vnstock-js';

const client = realtime.create({ symbols: ['VCB', 'FPT'] });

client.on('connected', () => console.log('Connected to SSI'));
client.on('quote', (q) => {
  console.log(`\${q.symbol} \${q.matched.price} @ \${q.totalVolume}`);
});
client.on('error', (err) => console.error(err));

client.connect();

// Later...
client.subscribe(['ACB']);
client.unsubscribe(['VCB']);
client.disconnect();
\`\`\`
```
