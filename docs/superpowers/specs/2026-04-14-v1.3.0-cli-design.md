# v1.3.0 Plan 3 — CLI Tool Design Spec

**Date:** 2026-04-14
**Parent spec:** [2026-04-10-vnstock-js-v1.3.0-design.md](2026-04-10-vnstock-js-v1.3.0-design.md) Section 6

## Context

CLI là thin layer trên core SDK — không có business logic riêng. Mục tiêu: mở rộng tệp user từ "JS/TS dev" sang "terminal user" để viết blog "10 lệnh terminal cho nhà đầu tư VN" sau release.

## Scope v1.3.0

### In

**4 commands** (lookup-focused, high-value):
- `vnstock quote <SYMBOL>` — snapshot 1 mã
- `vnstock history <SYMBOL>` — lịch sử OHLCV
- `vnstock search <QUERY>` — tìm mã theo tên
- `vnstock symbols` — liệt kê mã

### Out (deferred to v1.3.1+)

- `vnstock calendar` — lower value qua CLI (dev check trong code)
- `vnstock export` — overlap với `--json | jq` + shell redirect
- `gold`, `market overview`, `compare`, `watchlist` — v1.4+

## Decisions

| # | Decision | Rationale |
|---|---|---|
| D1 | 4 commands only cho v1.3.0 | YAGNI — calendar/export low ROI, release nhanh |
| D2 | Deps: `commander`, `cli-table3`, `picocolors` | Mature, stable, ~200KB tổng |
| D3 | Distribution: cả `npm i -g` và `npx` | Free với cùng `bin` field |
| D4 | Conditional init per command | `search`/`symbols` cần data; `quote`/`history` không |
| D5 | Default output: table + color (TTY), auto plain khi pipe | Match spec section 6.4 |
| D6 | Test strategy: unit test handler pure function | CLI thin, core đã test |
| D7 | Relative dates: `7d`, `1w`, `1m`, `1y` + absolute YYYY-MM-DD | Ergonomic |
| D8 | `--range` shortcut = `--from X --to today` | Gọn cho history |

## UX — Output formats

### `quote VCB`

```
VCB  Vietcombank · HOSE
59.3k  +0.1 (+0.17%)  KL 3.83M  Trần/Sàn 63.3/55.3
```

Màu: giá + %change theo VN convention (xanh tăng, đỏ giảm, vàng TC, tím trần, xanh dương sàn).

Verbose (`-v`): thêm NN mua/bán, cao/thấp, giá trị.

### `history VCB --range 7d`

```
Date        Close  Change  Volume
2026-04-14  59.3   +0.17%  3.83M
2026-04-11  59.2   +0.51%  4.10M
2026-04-10  58.9   -0.34%  3.95M
...
```

Verbose: thêm Open, High, Low.

### `search vina`

```
VNM  Vinamilk         HOSE
VND  VNDirect         HOSE
VHM  Vinhomes         HOSE
```

Default limit 10. `--limit N` override.

### `symbols --exchange HOSE`

Default — grid compact (symbol only, auto-fit terminal width):
```
VCB  VIC  VNM  VHM  FPT  MBB  ACB  HPG  MSN  VRE
VCI  SSI  STB  TCB  CTG  BID  VPB  TPB  SHB  LPB
...
```

Verbose (`-v`) — 2 cols:
```
VCB  Vietcombank
VIC  Vingroup
VNM  Vinamilk
...
```

Default limit 50. `--limit N` override.

## Relative date parsing

| Input | Meaning |
|---|---|
| `7d` | 7 days ago |
| `1w` | 1 week ago |
| `1m` | 1 month ago |
| `1y` | 1 year ago |
| `2026-01-01` | absolute |

Parser: regex `^(\d+)([dwmy])$` → compute from today. Invalid → exit code 1 with error message.

`--range 7d` expands to `--from 7d --to today`. Can't combine with `--from`/`--to`.

Default if no flags: `--range 30d`.

## Global flags

| Flag | Behavior |
|---|---|
| `--json` | Output JSON, suppress table/color |
| `--csv` | Output CSV, suppress table/color |
| `--no-color` | Disable ANSI color (still table) |
| `--quiet` | Suppress non-error logs (spinner, init message) |
| `-v`, `--verbose` | Per-command, shows more columns |
| `--help`, `-h` | Commander auto-generates |

Non-TTY stdout → auto `--no-color`, auto suppress spinner.

## Architecture

```
bin/vnstock (#!/usr/bin/env node)
  └─ require('../dist/cli/index.js')
       └─ src/cli/index.ts:
            ├─ commander setup
            ├─ route → command handler
            └─ handler calls core SDK
                 ├─ if command.requiresData: await init() (with spinner)
                 └─ call handleXxx({...}) → OutputData
                      └─ renderer.render(data, format)
```

### File layout

```
src/cli/
  index.ts           # commander bootstrap, route
  commands/
    quote.ts         # handleQuote(opts), pure function
    history.ts       # handleHistory(opts)
    search.ts        # handleSearch(opts)
    symbols.ts       # handleSymbols(opts)
  renderers/
    table.ts         # cli-table3 wrapper
    json.ts          # JSON.stringify
    csv.ts           # CSV writer
  format/
    date.ts          # parseRelativeDate(), formatDate()
    number.ts        # compactNumber() → "3.83M", formatPrice() → "59.3k"
    color.ts         # priceColor() using picocolors
bin/
  vnstock            # shebang + require dist/cli
```

### Handler signature

```typescript
export interface CommandContext {
  json: boolean;
  csv: boolean;
  color: boolean;
  quiet: boolean;
  verbose: boolean;
}

export interface QuoteOpts extends CommandContext {
  symbol: string;
}

export async function handleQuote(opts: QuoteOpts): Promise<string> {
  const data = await vnstock.stock.trading.priceBoard([opts.symbol]);
  if (opts.json) return renderJson(data[0]);
  if (opts.csv) return renderCsv([data[0]]);
  return renderQuoteTable(data[0], opts);
}
```

Pure function returns string. `bin/vnstock` prints to stdout, sets exit code. Makes testing trivial (no process spawn).

### `requiresData` flag

Each command module exports metadata:

```typescript
// src/cli/commands/quote.ts
export const meta = { requiresData: false };
export async function handleQuote(...) { ... }
```

`src/cli/index.ts` checks meta before command execution:

```typescript
if (meta.requiresData) {
  await withSpinner("Loading data…", () => init());
}
```

`init()` has disk cache so only first cold call is slow. Spinner uses `picocolors` (no `ora` dep — spec says keep deps minimal).

### Exit codes

- `0` — success
- `1` — user error (invalid symbol, bad date format, invalid arg)
- `2` — system error (network fail after init retry, SDK internal error)

## Testing

### Unit tests (majority)

`__tests__/cli/handleQuote.test.ts`, `handleHistory.test.ts`, etc.

Each test calls handler directly with mocked SDK:

```typescript
jest.mock("../../src", () => ({
  stock: {
    trading: { priceBoard: jest.fn().mockResolvedValue([{ symbol: "VCB", ... }]) },
  },
}));

it("returns formatted JSON when --json", async () => {
  const out = await handleQuote({ symbol: "VCB", json: true, ... });
  expect(JSON.parse(out).symbol).toBe("VCB");
});
```

### Date parser tests

`__tests__/cli/date.test.ts` — table-driven, test all formats + invalid cases.

### Renderer snapshot tests

`__tests__/cli/renderers.test.ts` — snapshot for table output (golden files for each command × default/verbose).

### Integration (1 smoke test)

`scripts/smoke-cli.sh` (not in Jest) — manual step before release:
```
node bin/vnstock quote VCB
node bin/vnstock quote VCB --json
node bin/vnstock search ngân hàng --limit 5
node bin/vnstock history VCB --range 7d
node bin/vnstock symbols --exchange HOSE --limit 10
```

Verify exit 0 + output looks right. Not in CI (needs network + real API).
